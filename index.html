    jChat (Originally jShare)

    I started off jShare with plans to allow users connected to the server to share files or packages of files with anyone connected, well it ended up turning into a multi-user chat program.

    Usage

    The intended use of jChat is anytime you want to chat with someone(s) running your own private server, with no chat logs so everything you say stays private.

    Features

    Implemented

    +Multi-user chat +Email based user names +Client GUI themes +Gravatar based avatars(Automatically grabbed using users email) +Color coded user names(Each new user is given a random color, making it easy to read and differentiate clients) +Online user list +Auto-updating client +New message notifications

    Planned

    +File sharing +Better notifications +User registration +Login using Google/Facebook/OpenId

    +Private messaging
